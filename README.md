## Project Description

Clone Coding Vocabulary List

## v1.0.0 (2022.09.22 ~ 2022.10.02)

-   단어 입력 폼
    -   이전 데이터 불러오기
    -   이전 데이터 삭제하기
    -   단어 수량 선택하기 (10개, 30개, 50개)
-   단어 카드
    -   카드 형식으로 단어 외우기
-   단어 테스트
    -   단어의 뜻을 입력하며 맞추기
